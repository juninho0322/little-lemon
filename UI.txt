===============================================
Colors — when to use
===============================================

Primary #F4CE14 (lemon): CTA accents, highlights, active nav pills. Text on it → #1F1F1F.

Secondary #495E57 (olive/green): Solid buttons, key sections, header accents. Text on it → #EDEFEE.

Surface #FFFFFF: Page background, cards.

Surface-soft #EDEFEE: Hovers, subtle sections, table rows.

Text #1F1F1F: Main body text.

Text-muted #5C5C5C: Helper/secondary text, captions.

Border #E3E3E3: Dividers, card borders, input outlines.

Coral #EE9972 / Blush #FBDABB / Leaf #6BA368: Optional accents (promo, badges, success).

Focus #2A9D8F: :focus-visible outline.
